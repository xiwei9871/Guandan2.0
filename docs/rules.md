# 掼蛋游戏规则

## 基本规则

### 玩家配置
- 4名玩家，分为两队
- 红队：南、北
- 蓝队：东、西
- 使用两副扑克牌（108张）

### 发牌
- 每人27张牌
- 游戏从2开始打起
- 级牌在每局游戏中变化

## 牌型说明

### 基本牌型
1. **单张**: 任意一张牌
2. **对子**: 两张相同的牌
3. **三张**: 三张相同的牌
4. **三带一**: 三张相同牌 + 一张任意牌
5. **三带二**: 三张相同牌 + 一对
6. **顺子**: 五张或以上连续单牌（不包括2和王）
7. **连对**: 三对或以上连续对子（不包括2和王）
8. **炸弹**: 四张相同牌
9. **王炸**: 两张王
10. **同花顺**: 五张或以上花色相同的连续牌

### 牌型大小
1. 王炸 > 同花顺 > 炸弹 > 其他牌型
2. 同花顺比较主牌大小
3. 炸弹比较牌面大小
4. 其他牌型必须牌型相同且主牌更大

## 特殊规则

### 级牌
- 当前等级的牌为级牌
- 例如：打2时，2为级牌
- 级牌在游戏中可以作为任意牌使用（逢人配）

### 逢人配
- 红桃级牌为"逢人配"
- 可以作为任意牌使用
- 可以与任何牌组成合法牌型
- 使用时需要明确声明其代表的牌

### 进贡规则
- 上一局最末游玩家向头游玩家进贡
- 进贡的牌必须是手中最大的牌
- 接收进贡的玩家需要返还一张牌

## 游戏流程

### 1. 准备阶段
- 玩家加入房间
- 点击准备按钮
- 4人全部准备后可以开始游戏

### 2. 发牌阶段
- 系统自动发牌
- 每人27张
- 玩家可以查看自己的手牌

### 3. 出牌阶段
- 从第一个玩家开始出牌
- 后续玩家必须出相同牌型且更大的牌
- 如果不想出或没有更大的牌，可以选择"不要"
- 三名玩家都"不要"后，出牌权回到最后出牌的玩家

### 4. 结算阶段
- 玩家出完所有牌的顺序决定名次
- 第一个出完的为头游
- 第二个出完的为二游
- 第三个出完的为三游
- 最后一个出完的为末游

## 计分规则

### 基础分
- 头游：+3分
- 二游：+2分
- 三游：+1分
- 末游：0分

### 队伍得分
- 同队两名玩家得分相加
- 得分高的队伍获胜

### 升级规则
- 获胜队伍的等级提升
- 失败队伍等级不变
- 如果同队包揽头游和二游，额外升一级

## 禁止行为

1. **出牌超时**: 每次出牌有时间限制
2. **非法出牌**: 不符合规则的牌型
3. **恶意退出**: 游戏中途退出
4. **通讯干扰**: 故意断开连接

## 游戏术语

| 术语 | 解释 |
|------|------|
| 头游 | 第一个出完牌的玩家 |
| 二游 | 第二个出完牌的玩家 |
| 三游 | 第三个出完牌的玩家 |
| 末游 | 最后一个出完牌的玩家 |
| 逢人配 | 红桃级牌，可作任意牌使用 |
| 级牌 | 当前等级的牌 |
| 炸弹 | 四张相同的牌 |
| 王炸 | 两张王同时出 |
| 进贡 | 末游向头游贡献最大的牌 |

## 示例场景

### 场景1：出顺子
- 玩家A出：3-4-5-6-7（五张顺子）
- 玩家B可以出：4-5-6-7-8（更大的顺子）
- 玩家B不能出：3-4-5-6-8（不是顺子）

### 场景2：出炸弹
- 玩家A出：4个5（炸弹）
- 玩家B可以出：4个6（更大的炸弹）
- 玩家B不能出：4个4（更小的炸弹）

### 场景3：使用逢人配
- 当前打2，红桃2为逢人配
- 玩家手中有：红桃2、黑桃3、红桃3、梅花3、方块3
- 可以出：红桃2（当作3）+ 其他4个3 = 5个3（炸弹）

## 技巧与策略

### 开局策略
1. 观察队友和对手的出牌习惯
2. 保留大牌作为控制手段
3. 尽早出掉小牌和散牌

### 中局策略
1. 根据场上形势调整策略
2. 与队友配合控制局面
3. 合理使用逢人配和炸弹

### 收尾策略
1. 计算剩余牌数
2. 保留出牌权
3. 争取队友接应

## 常见问题

### Q: 逢人配可以如何使用？
A: 逢人配（红桃级牌）可以作为任意牌使用，可以与其他牌组成任何合法牌型。

### Q: 如果没有更大的牌怎么办？
A: 可以选择"不要"，跳过本轮出牌。

### Q: 多少个玩家可以开始游戏？
A: 必须4名玩家全部准备才能开始游戏。

### Q: 游戏如何结束？
A: 当所有玩家都出完牌后，根据出完顺序计算名次和得分。

### Q: 如何升级？
A: 获胜队伍在下一局游戏中等级提升，失败队伍等级不变。
