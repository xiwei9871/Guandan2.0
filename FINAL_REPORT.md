## 部署建议

### 开发环境
```bash
npm run dev
# 访问 http://localhost:3000
```

### 生产环境
```bash
npm run build
npm start
# 或使用 PM2
pm2 start npm --name "guandan" -- start
```

### 环境变量
创建 `.env.local`:
```env
NODE_ENV=production
PORT=3000
ALLOWED_ORIGINS=https://yourdomain.com
```
